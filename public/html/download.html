The Easy Way recommended

Do you want to start using omp4j as fast as possible without any struggle? Simple follow the steps below and get started in few seconds.

  1. Make sure that a common JDK is installed (OracleJDK 7/8 or OpenJDK 6/7)
  2. Download one of the listed JARs below (latest version suggested)
  3. Create a shell alias for omp4j in form of java -jar <path-to-JAR>
  4. Learn how to use omp4j in the tutorial!

The Hard Way for hackers

Do you want to hack omp4j? Why not! Follow the steps below in order to checkout the source code and compile it by yourself. No rocket science!

  1. Make sure that a common JDK is installed (OracleJDK 7/8 or OpenJDK 6/7)
  2. Make sure that all following programs are installed - Scala2.9.2+, sbt0.13.5+, git
  3. Clone the repository via git clone git@github.com:omp4j/omp4j.git --recursive
  4. cd into src/main/java and execute update.sh
  5. cd back to project root and invoke sbt
  6. Now you may use sbt commands such as compile, run, test and assembly
