Online Demo try it out without installation

Do you need to try omp4j before installation? No problem! Simply insert your code to the left area and click translate button. The preprocessed code will appear in the right area. That's it! Have fun!

Please note that the online code transformation is employed as a demo only and isn't meant as a public cloud. The computation time is limited and only one file is supported. Also public classes are forbidden.

omp4j is an open-source project that generates no profit. We host this webpage at heroku using their free plan. Therefore only limited computation power is provided. We kindly ask you to use the demo for basic examples only. Please don't forget that other uses may want to try it as well. For common usage and complicated source code or multiple-file support, you may simply download the preprocessor and use it locally as you wish. Thank you for your courtesy.
