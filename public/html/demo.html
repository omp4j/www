Demo try it out online and for free

Need to try omp4j before installation? No problem! Simply insert your code to the left area and click translate button. The preprocessed text will appear in the right area. Easy as breathing. Have fun!

Please note that the online code transformation is employed as a demo only and isn't meant as a public cloud. The computation time is limited and only one file is supported. Also public classes are forbidden. You are asked to try only basic examples and share the computation power with other users. For more complicated resources simply download the preprocessor and use it locally. Thank you.
