Example usage in 10 seconds

Let us show you how simple may the Java parallel code become using omp4j. It won't take more than 10 seconds and it's definitely worth it, we promise. Just take a look at the following two examples:

Example #1 parallel invocation

Simply type a comment in front of a code block and let the preprocessor do the parallel invocation for you. Easy, isn't it? As you may see, the source code remains serial. That's what makes omp4j so powerful. Develop serial code - run it in parallel. Awesome! 

// omp parallel threadNum(4)
{
	System.out.print("hello ")
}

/* output */
hello hello hello hello

Example #2 for-loop in parallel

Well, invoking a block of code in parallel doesn't seem to be that useful, does it? What about for-loop parallel execution? Yes, it is that simple! Let the for-loop run in parallel only with a simple comment before the loop itself. Can you think of a simpler solution? We don't think so :) 

// omp parallel for
for (int i = 0; i < 10; i++) {
	System.out.print(i)
}

/* output */
2 0 6 7 1 9 5 3 4 8
