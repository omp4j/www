omp4j WWW presentation
======================

This is the main repository of https://www-omp4j.rhcloud.com. Due to RedHat OpenShift PaaS hosting, this repository contains also binaries.

Used technologies
-----------------
- Typesafe Activator
- Play Framework (v2.3)
- Scala
- AngularJS
- Twitter Bootstrap (v3)
- RedHat OpenShift PaaS
